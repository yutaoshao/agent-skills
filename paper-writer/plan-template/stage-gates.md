# Stage Gates

> Track paper lifecycle progress. Prevent skipping stages.

## Project

- Project name:
- Last updated:
- Current stage:

## S1 Project Analysis

- [ ] Repository structure explored
- [ ] README, docs, and key results reviewed
- [ ] Main contribution identified and confirmed with user
- [ ] Existing citations in codebase found
- [ ] Code artifacts mapped to paper sections

## S2 Literature Review

- [ ] Search strategy defined (keywords, databases)
- [ ] Literature search completed
- [ ] Papers organized by theme
- [ ] Research gaps identified
- [ ] Related work section drafted

## S3 Paper Structure

- [ ] One-sentence contribution defined and confirmed
- [ ] Narrative Principle applied (What/Why/So What)
- [ ] Paper outline generated
- [ ] Figure 1 concept sketched
- [ ] Target venue requirements reviewed

## S4 Writing Core

- [ ] Abstract drafted (5-sentence formula)
- [ ] Introduction drafted (contribution bullets, 1-1.5 pages)
- [ ] Methods section drafted (enable reimplementation)
- [ ] Experiments section drafted (explicit claims)
- [ ] Related Work section drafted (methodological organization)
- [ ] Limitations section drafted
- [ ] Conclusion drafted

## S5 Figures & Tables

- [ ] All figures generated (450 DPI, colorblind-safe)
- [ ] Figure captions self-contained
- [ ] TikZ diagrams verified against paper text
- [ ] Table data cross-checked against source files
- [ ] Bold/underline formatting correct for best/second-best

## S6 Citation Management

- [ ] All citations verified (not hallucinated)
- [ ] No placeholder citations remaining
- [ ] Citation gaps addressed (temporal, methodological, foundational)
- [ ] Target reference count met
- [ ] BibTeX entries complete (volume, pages, DOI)

## S7 Polish

- [ ] De-AI patterns removed (English and/or Chinese)
- [ ] Three-round quality check completed
- [ ] Style check script passed
- [ ] Writing reads naturally, not mechanically

## S8 LaTeX Compilation

- [ ] Template correctly set up
- [ ] Compiles without errors
- [ ] No undefined references or citations
- [ ] Typography checklist passed
- [ ] Within page limit

## S9 Review & Submission

- [ ] Conference-specific checklist completed
- [ ] Self-review from reviewer perspective done
- [ ] Anonymization verified (for blind review)
- [ ] Submission package prepared (cover letter, statements)
- [ ] Final PDF reviewed

## S10 Revision & Resubmission

- [ ] Reviewer response matrix created
- [ ] Each comment addressed
- [ ] Change log generated
- [ ] Revised paper compiled and verified
- [ ] Camera-ready version prepared (if accepted)
